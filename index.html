⚙️ Connessione Google Sheets

✓ Salvato!

1 Struttura Google Sheet

Crea un foglio Google con questi 7 tab (nomi esatti):

PRODOTTI   FORNITORI   CLIENTI   ACQUISTI   VENDITE   MOVIMENTI   CONFIG

2 Intestazioni – riga 1 di ogni foglio

PRODOTTI

ID | Nome | Categoria | Unita_misura | Prezzo_std_acq | Prezzo_acq_vend

FORNITORI

ID | Nome | Tipo | Email | Telefono | Citta | Indirizzo | Provincia | Regione | IBAN | Note

CLIENTI

ID | Nome | Tipo | Email | Telefono | Citta | Indirizzo | Provincia | Regione | IBAN | Note

ACQUISTI

ID | Data | ID_Fornitore | Flag_B | Pagato | Data_Pagamento | Note

VENDITE

ID | Data | ID_Cliente | Flag_B | Pagato | Data_Pagamento | Note

MOVIMENTI

ID | ID_Acquisto | ID_Vendita | ID_Prodotto | Tipo | Quantita | Prezzo_unitario | Subtotale

→ ID_Acquisto e ID_Vendita contengono il padre (solo uno riempito) • Tipo = Acquisto o Vendita

3 Condividere e ottenere API Key

1. Condividi il foglio come Chiunque abbia il link → Visualizzatore

2. Copia l'ID dal link (stringa tra /d/ e /edit)

3. console.cloud.google.com → Crea progetto → API e servizi → Credenziali → Crea Chiave API

4. Abilita Google Sheets API dalla libreria

5. Incolla ID e Key qui sopra e clicca Salva

4 GitHub Pages

1. Crea repository su github.com → carica questo index.html

2. Settings → Pages → Branch: main → /root → Salva

3. Dopo ~2 min disponibile su https://tuonome.github.io/nome-repo

5 Scrittura su Google Sheets (Apps Script)

Nel tuo Google Sheet: Estensioni → Apps Script. Incolla questo codice:

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const ss = SpreadsheetApp.getActiveSpreadsheet();
    const sheet = ss.getSheetByName(data.sheet);
    sheet.appendRow(data.row);
    return ContentService.createTextOutput(
      JSON.stringify({ok: true})
    ).setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(
      JSON.stringify({ok: false, error: err.message})
    ).setMimeType(ContentService.MimeType.JSON);
  }
}

Poi: Distribuisci → Nuova distribuzione → App web → Chiunque → Distribuisci. Copia l'URL e incollalo come Script URL nel box di config sopra.
