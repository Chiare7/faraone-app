⚙️ Configurazione Google Sheets API

✓ Salvato!

1 Crea il Google Sheet

Vai su sheets.google.com e crea un nuovo foglio. Rinominalo "Faraone DB".

Crea 5 fogli con questi nomi esatti (clic su + in basso):

PRODOTTI    FORNITORI    CLIENTI    ACQUISTI    VENDITE

2 Struttura i fogli (intestazioni)

PRODOTTI – riga 1:

ID | Nome | Categoria | Prezzo Acquisto | Prezzo Vendita | Quantità Stock

FORNITORI / CLIENTI – riga 1:

ID | Nome | Email | Telefono | Indirizzo | IBAN | Note

ACQUISTI / VENDITE – riga 1:

ID | Data | ID Controparte | Prodotto | Quantità | Prezzo Unitario | Totale | Flag B | Pagato | Data Pagamento | Note

3 Rendi il foglio pubblico in lettura

Clic su Condividi → Chiunque abbia il link → Visualizzatore. Copia l'ID dal link URL (la stringa lunga tra /d/ e /edit).

4 Ottieni la API Key Google

Vai su console.cloud.google.com → crea un progetto → API e servizi → Credenziali → Crea credenziali → Chiave API.

Poi abilita Google Sheets API dalla libreria API.

Incolla l'ID del foglio e la API Key nel box sopra e clicca Salva Configurazione.

5 Pubblica su GitHub Pages

1. Crea un repository su github.com (es: faraone-app)

2. Carica il file index.html

3. Vai su Settings → Pages → Branch: main → /root → Salva

4. Dopo qualche minuto l'app sarà disponibile su https://tuonome.github.io/faraone-app

6 ⚠️ Nota sulla scrittura dati

L'API Key pubblica permette la sola lettura. Per la scrittura (inserimento nuove righe) vengono usati gli Apps Script di Google. Dopo il setup base, segui le istruzioni per creare uno script che riceve le richieste POST e scrive sul foglio. Le istruzioni dettagliate sono nella guida allegata.
